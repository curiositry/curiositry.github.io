About

Automathic.org is a crowd-sourced repository of Mathematical Formulas, Rules, and Equations contributed by people like you. All the information you need and nothing more â€“ neatly organized, appealingly presented, and completely free.

Technology

This was one of the first forays I made into serious web development. I have not open-sourced it yet because the code is a mess, but I intend to when I have a chance to clean it up a bit. Automathic is built on PHP/MySQL.

It uses a good bit of third-party code, including PHPMarkdown, a user system written by Photosythesis, Hint.css, Mathjax, Jquery, and Entypo Pictograms. But not Bootstrap.
