Automathic.org

About

This was one of the first forays I made into serious web development. I have not open-sourced it yet because the code is a mess, but I intend to when I have a chance to clean it up a bit.

Technology

It is built on PHP/MySQL with as little JavaScript as possible.

It uses a good bit of third-party code, including PHPMarkdown, a user system written by Photosythesis, Hint.css, Mathjax, Jquery, and Entypo Pictograms. It doesn’t use Bootstrap.

Typeset in: Linux Libertine SC & Ubuntu.
