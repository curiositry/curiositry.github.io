Bohemian: Responsive Pelican Theme

Mon 31 March 2014


About

Source Code

This is a pelican theme I wrote to use here on my github page, replacing my older Clean theme.

Bohemian is Free, Responsive, Open Source, & Typographically Tasteful. Built to provide a great reading experience with minimal clutter, Bohemian will ready your Pelican site for 21st century.

Features

  • Fully Responsive
  • Lightweight
  • Typeset in the beautiful Alegreya, served by Brick for blazing speed.
  • Two Handsome Subthemes build on the same base â€¦ the one on this site and the theme used by Science Class
  • Modern, clutter free design with an emphasis on a great reading experience.
  • Option to use text or image for the logo

Technology

Bohemian makes use of the wonderful Pelican, Python, Typeplate.css, Pygments syntax highlighting, and Brick webfonts.
